# 🚗 Audi India Clone Website

A front-end clone of the official Audi India website built using HTML & CSS.

🔗 **Live Demo:** https://aneeshkumaryadav.github.io/audi-india-clone/

---

## 📌 About The Project

This project is a static front-end clone of the Audi India website.  
It replicates the homepage layout, navigation bar, car sections, and footer.

The goal of this project was to practice:
- Website structuring
- Layout design
- CSS styling
- Deployment using GitHub Pages

---

## 🛠️ Tech Stack

- HTML5
- CSS3

---

## ✨ Features

- Navigation bar
- Car model showcase section
- Hero banner section
- Footer with links
- Clean UI design
